# Linguagens para Internet I - 2025

## Sobre a disciplina

Este repositório reúne as atividades desenvolvidas na disciplina **Linguagens para Internet I**, cursada em **2025**.

O objetivo da disciplina é apresentar os conceitos fundamentais do desenvolvimento web, abordando a criação de páginas e aplicações para a internet por meio das principais tecnologias utilizadas atualmente.

## Conteúdos estudados

### HTML

O **HTML (HyperText Markup Language)** é a linguagem responsável pela estruturação das páginas web. Com ele, são definidos elementos como títulos, parágrafos, listas, tabelas, formulários e demais componentes que organizam o conteúdo exibido no navegador.

### CSS

O **CSS (Cascading Style Sheets)** é utilizado para estilizar as páginas desenvolvidas em HTML. Por meio dele, é possível definir cores, fontes, espaçamentos, layouts, animações e tornar a interface mais agradável e responsiva.

### JavaScript

O **JavaScript** é a linguagem de programação responsável por adicionar interatividade às páginas web. Com ela, é possível manipular elementos da página, validar formulários, responder a eventos do usuário e criar aplicações dinâmicas.
